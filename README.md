# Racer

A top-down arcade racing game prototype focused on responsive handling, and time-trial gameplay.

## Features

- Arcade-style car physics
- Skid marks and camera look-ahead
- Lap timing with deltas and best laps
- Countdown start and finish state
- Grass & water slowdown and wall collisions
- Camera lag & look-ahead
- Tile-based tracks

## Controls

- Arrow Left / Right - Steering
- Space - Throttle
